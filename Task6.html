HTML vs HTML5

HTML HTML5
HTML stands for Hypertext Markup Language. HTML5 stands for Hypertext Markup Language, version 5.
HTML4 was the previous version, and it had limited support for multimedia elements. HTML5 introduced native support for audio and video elements, making multimedia integration easier.
HTML4 used presentational elements and attributes for styling and layout. HTML5 encourages the use of CSS for styling, separating content from presentation.
